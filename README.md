# CCT Sprint Dashboard 💰

Common Cents Tania — 90-Day Relaunch Sprint Dashboard  
**June 1 – August 29, 2026**

## What This Does
- Live sprint progress tracker (Day X of 90)
- Daily pace checker synced to Todoist
- Weekly content goal progress bars
- Community follower tracker
- Auto-researched trending keywords (TikTok, IG, Google)
- Weekly message reflection prompts

## Local Development
```bash
npm install
npm start
```
Opens at `http://localhost:3000`

## Deploy
Connected to Netlify — pushes to `main` branch auto-deploy.
